# 🚀 RTE Client — Next-Gen Client-Side Utility

**RTE Client** is a lightweight, optimized, and completely client-side mod for Fabric that consolidates dozens of useful interface, visual, and QoL (Quality of Life) enhancements into a single, cohesive utility tool. 

It does not alter server-side mechanics, is not a malicious cheat, and is designed purely to enhance gameplay comfort, customization, and visual appeal. ✨

---

### 🧩 How to Open the Menu?
By default, the built-in interactive **ClickGUI** opens using the **RightShift** key.

Inside the menu, you can toggle modules, configure their parameters, and rebind hotkeys to your liking with a single click. The interface and feature descriptions are fully localized in **English, Russian, and Chinese**.

---

### 🔥 Key Features
* 🎨 **Interface & HUD** — Custom on-screen indicators, improved tooltips, clean armor/status effect overlay bars, and smooth in-game chat animations.
* ⚙️ **Gameplay & QoL** — Smart tools to automate tedious tasks (seamless inventory management, auto-sprint, auto-reconnect, fast quit, and copying screenshots directly to your clipboard).
* 🕶️ **Visual Tweaks & NoRender** — Fixes for vanilla graphical bugs and a highly customizable toggle to disable performance-heavy or distracting render elements (fog, particles, fire overlay, camera shake) for a clean visual experience.
* 💎 **Exclusive Features**:
  * **Built-in Mechvibes** — An audio engine that plays satisfying mechanical keyboard clicks directly through Minecraft's native sound system as you type.
  * **Self-Hosted e4mc (Tunneling)** — Open your local LAN world to friends over the internet in a single click using a custom domain (`13377777.xyz`).
  * **In-Game Account Switcher** — A sleek profile manager integrated right into the main menu for hot-swapping between official Microsoft accounts and offline profiles without restarting the game.

---

### ⚠️ Performance Notice
The mod contains targeted micro-optimizations (such as **Dynamic FPS** — dropping rendering load to 15 FPS when minimized via Alt+Tab, and basic Frustum Culling). However, it **IS NOT** a substitute for dedicated performance engines. 

For the best possible FPS boost, it is highly recommended to use RTE Client alongside **Sodium**.

---

### 🛡️ Safety & Compatibility
* ✅ **100% Client-Side** — All modifications happen strictly on your end.
* ✅ **No PvP Hacks** — The mod contains no malicious, automated combat, or rage features.
* ✅ **Allowed Everywhere** — Thanks to its architecture, it is safe to use on most public servers, anarchies, and private vanilla SMP projects.

---

### ⚙️ Requirements
* 📦 **Minecraft:** 1.21.4
* 🧵 **Loader:** Fabric Loader
* 🛠 **API:** Fabric API

# 🚀 RTE Client — Next-Gen Client-Side Utility

Мод не изменяет серверные механики, не является вредоносным читом и создан исключительно для повышения комфорта, кастомизации и визуального восприятия игры. ✨

### 🧩 Как открыть меню?
По умолчанию встроенный интерактивный **ClickGUI** открывается на клавишу **RightShift (Правый Шифт)**. 

Внутри меню вы можете в один клик переключать модули, настраивать их параметры, а также переназначать бинды клавиш под себя. Интерфейс и описания функций полностью локализованы на **русский, английский и китайский языки**.

---

### 🔥 Главные особенности
* 🎨 **Интерфейс и HUD** — кастомные индикаторы на экране, улучшенные всплывающие подсказки, информативные панели состояния брони/эффектов и плавная анимация игрового чата.
* ⚙️ **Геймплей и QoL** — умные инструменты для автоматизации рутины (удобное управление инвентарем, автоспринт, автореконнект, быстрый выход и копирование скриншотов в буфер обмена).
* 🕶️ **Визуальные твики и NoRender** — исправление ванильных графических багов и гибкий тумблер отключения лишних эффектов рендера (туман, частицы, огонь на экране, тряска камеры) для чистой картинки.
* 💎 **Эксклюзивный функционал**:
  * **Аналог Mechvibes** — встроенный звуковой движок, воспроизводящий сочные клики механической клавиатуры прямо внутри игры.
  * **In-game Account Switcher** — удобное меню на главном экране для быстрого переключения между профилями без перезапуска игры.

Для достижения максимальной производительности настоятельно рекомендуется использовать RTE Client совместно с модом **Sodium**.
